Горячие клавиши оператора

Горячие клавиши ускоряют работу оператора — меньше кликов, больше диалогов в час.

Управление диалогами

Alt + AПринять следующий диалог из очереди
Alt + DЗакрыть текущий диалог
Alt + TПередать диалог другому оператору
Alt + NДобавить внутреннюю заметку
Alt + LДобавить тег к диалогу

Навигация

Alt + ↓ / ↑Следующий / предыдущий диалог в списке
Alt + 1Перейти во «Входящие»
Alt + 2Перейти в «Мои диалоги»
Alt + 3Перейти в «Все диалоги»
Ctrl + KБыстрый поиск по диалогам

Отправка сообщений

EnterОтправить сообщение
Shift + EnterПеренос строки в сообщении
/Открыть список быстрых ответов
Ctrl + BЖирный текст
Ctrl + IКурсив

💡 Горячие клавиши можно переназначить в Настройки → Кабинет → Горячие клавиши.
