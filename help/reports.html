Экспорт отчётов и данных диалогов

Все данные можно выгрузить для внешнего анализа, отчётов руководству или интеграции с BI-системами.

Виды отчётов

  • Сводный отчёт — диалоги, время ответа, CSAT за период
  • Отчёт по операторам — нагрузка и эффективность каждого сотрудника
  • Диалоги — полная история переписки с фильтрами
  • AI-статистика — сколько вопросов решил автопилот, процент передачи
  • Теги и темы — о чём чаще всего спрашивают клиенты

Экспорт данных

Перейдите в Аналитика → Отчёты → Экспорт:

  • CSV — для Excel, Google Sheets
  • XLSX — готовый Excel с форматированием
  • JSON — для разработчиков и BI-систем
  • API — автоматическое получение через REST API (тариф Pro)

Автоматические отчёты на email

Раздел Аналитика → Отчёты → Автоотправка:

  • Ежедневная сводка — каждое утро на email руководителя
  • Еженедельный отчёт — по понедельникам за прошлую неделю
  • Кастомное расписание и список получателей
