How to Add Live Chat to Your Website in 5 Minutes · Prochat

Step-by-step guide to installing a live chat widget on any website. Works with HTML, WordPress, Tilda, and any CMS. No developer needed.

Adding live chat to your website is easier than most people think. You don't need a developer. With Prochat, you can have a fully functional chat widget running in under 5 minutes.

Method 1: Plain HTML Website

  1. Sign up at prochat.uz (no credit card needed)
  2. Go to Settings → Install Widget → Copy the code snippet
  3. Paste the code just before the closing </body> tag in your HTML
  4. Save and refresh your site — the chat widget appears immediately

Method 2: WordPress

Two options: search for "Prochat" in the WordPress plugin directory, or use WPCode to insert the script globally in the footer. Works with every WordPress theme including Divi, Elementor, and Astra.

Method 3: Tilda

Go to Tilda Site Settings → More → HTML code before </body>. Paste the Prochat script. It will run on every page of your Tilda project automatically.

One Script, All Pages

You only install the script once. Prochat's widget runs across every page of your site automatically — no need to add it page by page.

⚡ After installing, spend 10 minutes customizing: set your brand color, upload an operator photo, and write a greeting message. This increases chat engagement by 15–20%.

Verify It Works

  1. Open your website in a new browser window (or incognito)
  2. Look for the chat button in the bottom-right corner
  3. Send a test message
  4. Check that it appears in your Prochat operator panel
